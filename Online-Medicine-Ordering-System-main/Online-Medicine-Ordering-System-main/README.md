# 💊 MediQuick – Online Medicine Ordering System

MediQuick is a modern and user-friendly **online pharmacy platform** that allows users to browse, search, and order medicines securely from the comfort of their homes. Designed with both **customer experience** and **admin efficiency** in mind, this system combines attractive UI with powerful backend processing.

## 🧰 Tech Stack

- **Frontend**: HTML, CSS, JavaScript, Tailwind CSS  
- **Backend**: Java (Servlets & JSP)  
- **Database**: MySQL  
- **Architecture**: MVC  
- **Tools**: Apache Tomcat, MySQL Workbench, Eclipse IDE

---

## ✨ Features Overview

- 🔍 **Browse Medicines**
- 🛒 **Add to Cart**, view total with real-time calculation
- 📝 **Order tracking** with multiple status stages (e.g., To Pay → Processing → Delivered)
- 💳 **Multiple Payment Options** (Card, PayPal, Bank Transfer)
- 🔐 Secure **User Registration & Login**
- 👤 Personal **Profile Management** (Edit / Update / Delete Profile)
- 📦 View **Order History**
- 📦 Full CRUD for **Medicines**
- 📊 Manage **Customer Orders**, track status
